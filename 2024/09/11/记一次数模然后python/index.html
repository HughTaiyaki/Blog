python基础

python是一种解释性语言，直接通过解释器运行，不需要编译链接。python中一切皆对象，给一个内容绑定一个标签就是变量名。编程风格呢，全大写一般表示常量。暂时想到的就是这些。

f - string

格式化字符串可以做到左对齐和右对齐，然后还有居中对齐。也可以进行字符截断。例如:

  • 精度截取print(f"{123.456789:.3f}") 会输出123.456
  • 分隔符print(f"{12345678:,}"),print(f"{12345678:_}") 会输出12,345,678和12_345_678
  • 显示进制print(f"{1234:x}")会输出4d2。如果想要进行进制转换，可以使用hex(1234), oct(1234)和bin(1234)。然后我们就能看到输出了0x4d2,0o2322和0b10011010010我们可以发现他们都是字符串。

类

python的类和别的语言相差不大

和C++ 构造函数类似，python使用特殊方法 __init__，在类实例化的时候自动调用；和C++this 指针类似，python使用self参数来引用当前对象。

C++使用 public、private 和 protected 来控制类成员的访问权限。Python默认所有的成员都是公有的，但可以通过使用单下划线 _ 和双下划线 __ 前缀来模拟保护和私有的行为。
