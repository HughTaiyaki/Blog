配置说明

一般在根目录下的_config.yml里面就能改语言标题和一些基础设置

  • yml 文件是 YAML（YAML Ain’t Markup Language）格式的文件，用于配置和数据序列化。YAML 是一种人类可读的数据格式，广泛用于配置文件和数据交换。
    • yaml基本语法
      yaml: key: value
      列表: 嵌套:
  • Hexo 使用 Node.js 编写，并通过 js-yaml 这样的库来解析 YAML 文件
  • Hexo 使用 Node.js 解析 YAML 文件，并将其转换为 JavaScript 对象，以便在程序中使用。通过正确配置 _config.yml 文件，你可以轻松地自定义 Hexo 站点的各种设置。
