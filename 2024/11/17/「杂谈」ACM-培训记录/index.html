一：Number Theory

这部分主要是一些数论知识，设计蛮多随机算法的，对应 oi- wiki 的这部分 https://oi-wiki.org/math/number-theory/sieve/

A 题：求 $[n^2 ,n^2 + n]$ 的素数个数。 分段筛法（优化的埃式筛）

⭐️ 【A 题】

C 题：n 的范围 [1,1000] ，a_i 的范围 [1, 10^18]，判断素数

⭐️ 【C 题】

  • 使用 Miller-Rabin 素性测试，这是一种概率性算法，可以高效判断大数的素性。

  • 为了避免溢出：

    • 使用 __int128 类型（如果编译器支持）

      • 或者用二进制分解实现模乘法（Binary Multiplication），避免大数直接相乘。

    • Miller-Rabin 素性测试对每个数的复杂度为 O(klog⁡3n)，其中 k 是测试的轮数，通常取 k=5 足够准确。

D 题：给定 n 和 m，找到 1 <= x <= n 中与 m 互质gcd(x,m) = 1 的数字个数，如果 x 和 m 的 gcd 为 1，表示它们没有公因子。

⭐️ 【D 题】

大致思路：分解 m （试除法），枚举子集（位操作），然后计算与 m 不互质的数（容斥原理）

F 题 : 给定一个数 N <= 9*10^18，找出大于 1 的自然数 x 和 y，使得 N = x * y
大致思路：判断是否是素数（Miller-Rabin 素性测试），不是素数用Pollard-Rho因式分解 找到 N 的一个非平凡因子 x
