Summary

Lab 0 主要包含的内容为终端安装 （Windows Power Shell/GIT）、Python安装、文本编译器安装（课程官方推荐使用 Visual Studio Code (VS Code)）、终端基本操作、利用终端做Test等内容。

  • 然后我就发现了翻译版的textbook，🙇‍感谢大佬们，附上链接

    https://github.com/wizardforcel/sicp-py-zh

所以我就不自己写笔记了，总结一些对自己有启发的东西吧

  • =号是实现抽象的的一种简单方式
  • python中，解释器必须维护某种内存来跟踪名称的绑定，这些内存称为环境。python内存管理是不同于C语言的，c的内存管理是一种堆栈的模式。
  • 表达式树(expression tree)是分析函数的的一种数据结构，有点像编译原理的语义分析。（没学过编译原理）
  • 函数调用的时候才产生framew

一年后的今天

现在再看真的学了好多东西啊

  • 第一章的 py 里函数是一等公民，可以用高阶函数，函数传函数；
  • 第二章的面向对象，继承，多重继承；数据的抽象（数据结构）；局部变量；泛用方法，就是爪哇的抽象函数；
  • 第三章涉及到编译处理，递归函数的深度理解；异常；
  • 第四章分布式和并行计算
  • 第五章涉及协程

UCB不愧是UCB，入门课的质量真不戳，学的很爽！
